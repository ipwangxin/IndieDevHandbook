# 国内独立开发冷启动手册

一本写给国内独立开发者的免费实践手册。

它整理从选 idea、定义产品、AI / vibe coding 开发、MVP、部署上线、支付、SEO、推广、流量联盟到放弃复盘的完整路径。重点不是讲成功学，而是讲一个小产品怎么更快拿到真实反馈。

## 在线阅读

正式站点：

```text
https://book.swep.top/
```

如果你正在做独立开发项目，也可以免费加入流量联盟：

```text
https://ally.swep.top/
```

## 适合谁

- 正在做第一个独立开发项目的人。
- 已经上线但没有第一批用户的人。
- 想用 AI / vibe coding 快速做 MVP 的人。
- 发帖有收藏和点赞，但没有转化的人。
- 正在纠结部署、支付、定价、SEO 和推广的人。
- 有项目暂停、放弃、增长停滞，想做复盘的人。

## 本地运行

```bash
npm install
npm run docs:dev
```

构建静态站点：

```bash
npm run docs:build
```

预览构建结果：

```bash
npm run docs:preview
```
