# 国内上线合规

> *字数：约 1734 字；预计阅读时间：4 分钟。*


这一章整理国内网站、小程序和 App 常见门槛。具体要求会随平台和监管规则变化，最终以官方后台和接入服务商要求为准。

## 国内网站

### ICP 备案

如果网站使用中国境内服务器并向用户提供互联网信息服务，通常需要做 ICP 备案。

```text
域名解析到中国境内服务器 -> 通常需要 ICP 备案
```

备案不是经营许可证。它更像是网站上线前的基础登记。

常见材料包括：

- 域名
- 云服务器
- 主体信息
- 负责人信息
- 网站名称和服务内容
- 平台要求的核验材料

### ICP 经营许可证

ICP备案和 ICP 经营许可证不是一回事。

- 非经营性互联网信息服务：通常是 ICP 备案。
- 经营性互联网信息服务：可能需要 ICP 经营许可证。

“经营性”不是简单等于“网站上有钱”，实际判断会涉及服务类型、收费方式、业务模式和主管部门口径。不确定时，应咨询云厂商备案顾问、当地通信管理局或专业合规服务。

### 公安备案和其他事项

部分网站上线后还可能涉及公安联网备案、隐私政策、用户协议、Cookie 提示、内容审核等。

如果涉及医疗、金融、教育、新闻、直播、社交、生成式 AI、用户发布内容、未成年人、数据跨境，合规复杂度会明显上升。

## 微信小程序

小程序不是“随便传一个网页”。常见门槛：

- 注册小程序账号
- 选择主体
- 选择服务类目
- 主体认证
- 小程序备案
- 提交审核
- 配置服务器域名
- 如果收款，需要微信支付商户号

类目很重要。电商、医疗、教育、金融、社区、内容、社交类产品可能需要额外资质或更严格审核。

## 小程序主体差异

| 主体 | 常见能力 | 常见限制 | 适合场景 |
| --- | --- | --- | --- |
| 个人 | 注册门槛低，可做部分工具、内容、展示类小程序；满足条件后可能开通流量主 | 类目有限，商业能力受限，通常不适合完整交易闭环 | 个人作品、轻工具、内容展示、测试入口 |
| 个体工商户 | 有经营主体，可做更多商业类目，可申请微信支付 | 仍受类目资质和平台规则限制 | 小生意、轻电商、服务预约、工具收费 |
| 企业 | 能力最完整，可认证、可接微信支付、可申请更多类目和接口 | 需要营业执照、认证、财务和合规成本 | 正式商业产品、SaaS、交易、企业服务 |
| 政府/媒体/其他组织 | 特定组织能力和类目 | 资质要求明确，审核更严格 | 政务、媒体、公共服务等 |

个人主体不等于不能变现，但直接商业化能力通常更受限。常见方式是先用工具、内容或流量主验证，再根据业务需要切换到个体户或企业主体。

## 国内 App

国内 App 的常见事项：

- 应用市场开发者账号
- 应用包名和签名
- 隐私政策
- 权限说明
- 用户协议
- App 备案或相关备案要求
- 平台安全检测
- 部分市场可能要求软著
- 特殊类目需要额外资质

### App 备案和网站备案

App 备案和网站 ICP 备案都属于互联网信息服务备案体系里的事情，但对象不同。

```text
网站备案：网站域名和网站服务备案
App 备案：移动互联网应用程序备案，包括 App、小程序、快应用等形态
```

已履行网站备案手续的主办者，通常补充 App 有关信息，不需要重复填报主体真实身份信息。备案通常通过网络接入服务提供者或应用分发平台提交。

### 软著是什么

软著是计算机软件著作权登记，用来证明软件著作权权属和登记信息。

```text
软著 = 权属证明/版权登记
备案/许可证 = 上线经营相关合规
平台审核 = 分发平台准入
```

软著不是所有产品上线的通用许可证，也不是 ICP 备案或经营许可证的替代品。

## 国内安卓分发不是一个市场

国内 Android 上架和 Google Play 不一样。常见渠道包括华为、小米、OPPO、vivo、应用宝、百度、360、荣耀等市场。

每个市场都可能有自己的开发者账号、资质要求、隐私检测、安全检测、权限审核和类目规则。你可能要面对多个市场分别提交、分别审核、分别更新。

## 游戏和小游戏

游戏和普通工具 App 不一样，门槛通常更高。

国内游戏可能涉及：

- 游戏版号
- 防沉迷
- 未成年人保护
- 实名认证
- 内购和支付规则
- 广告合规
- 内容审核
- 平台审核

小游戏还要看微信、抖音、快手、QQ 等平台各自规则。

## 常见类目资质速查

| 类型 | 可能遇到的门槛 |
| --- | --- |
| 纯内容站 / 免费书 | 备案、版权、隐私政策、评论区内容管理 |
| 工具站 | 备案、隐私政策、第三方 SDK、文件上传安全 |
| AI 工具 | 生成内容管理、用户上传数据、模型服务条款、深度合成/生成式 AI 标识 |
| 电商 | 营业执照、支付、售后、商品资质、平台规则 |
| 教育培训 | 教育相关资质、课程内容审核、未成年人保护 |
| 医疗健康 | 医疗资质、健康建议边界、广告法和平台审核 |
| 金融理财 | 金融牌照、风险提示、投资建议合规 |
| 社区/UGC | 内容审核、举报、用户协议、实名或安全机制 |
| 游戏 | 版号、未成年人、防沉迷、支付规则 |
| 直播/语音房 | 内容安全、资质、实名、审核和平台规则 |
| 算命/风水/占卜/运势 | 迷信、诱导付费、虚假承诺、内容审核风险 |
| 招聘/撮合/交易平台 | 信息审核、资质、用户纠纷、平台责任 |

## 国内上线流程

### 国内网站

```text
买域名
-> 选择服务器所在地
-> 如果使用境内服务器，准备 ICP 备案
-> 部署网站
-> 配置 HTTPS
-> 添加隐私政策和用户协议
-> 接入统计和反馈入口
-> 如果收费，确认支付和资质
-> 提交搜索引擎收录
```

### 微信小程序

```text
注册小程序账号
-> 选择主体
-> 选择服务类目
-> 准备资质
-> 完成备案
-> 配置服务器域名
-> 开发和测试
-> 填写隐私和权限信息
-> 提交微信审核
-> 发布
-> 如果需要变现，申请流量主或微信支付
```

### 国内安卓市场

```text
准备应用包和签名
-> 准备开发者账号
-> 准备隐私政策和权限说明
-> 准备软著或其他资质，若市场要求
-> 完成安全检测
-> 分别提交到各应用市场
-> 处理各市场审核反馈
-> 维护多渠道版本更新
```
