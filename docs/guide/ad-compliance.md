# 广告接入和变现合规

> *字数：约 1578 字；预计阅读时间：4 分钟。*


广告是变现方式之一，不是所有产品都适合。接广告之前，先确认广告不会破坏核心体验，也不会和隐私、平台规则、用户信任冲突。

## App 可以通过哪些渠道接广告

App 广告通常不是自己直接找广告主，而是接入广告平台 SDK 或广告聚合平台。

### 海外常见渠道

- Google AdMob
- AppLovin MAX
- Unity Ads
- Unity LevelPlay / ironSource
- Meta Audience Network，具体可用性以平台政策为准
- Pangle，TikTok for Business 生态下的广告平台

这些平台通常提供 Banner、插屏广告、激励视频广告、原生广告、开屏广告，以及广告聚合和瀑布流/竞价。

### 国内常见渠道

- 腾讯优量汇
- 穿山甲
- 快手联盟
- 百度联盟
- 华为广告
- 小米广告联盟
- OPPO / vivo 等厂商广告平台

如果 App 主要在国内安卓市场分发，厂商广告平台和国内广告联盟会更常见。

## App 广告接入门槛

常见门槛：

- App 已经可正常运行。
- 有应用包名、签名和应用信息。
- 注册广告平台账号。
- 提供收款和税务信息。
- 接入广告 SDK。
- 配置隐私政策和权限说明。
- 按平台要求处理用户同意、数据收集和个性化广告。
- 遵守 App Store / Google Play / 国内应用市场的广告和隐私规则。

如果面向海外，通常还要关注 GDPR、CCPA/CPRA、ATT、Google Play Data safety、App Store Privacy Nutrition Labels。

## App 广告怎么算收益

广告收益通常不是简单按“用户数”算，而是看：

```text
广告展示次数
有效展示
eCPM
填充率
点击率
广告类型
用户地区
用户价值
广告平台分成
```

最常见公式是：

```text
预估收入 = 有效曝光次数 / 1000 × eCPM
```

Banner 打扰小但 eCPM 通常较低；插屏收益较高但容易打断体验；激励视频适合用户主动换权益的场景；原生广告适合信息流和内容场景。

## 小程序怎么接广告

微信小程序主要通过“流量主”接入官方广告组件。

常见流程：

```text
小程序上线
-> 满足流量主开通条件
-> 在微信公众平台开通流量主
-> 补充财务资料
-> 创建广告位
-> 获取广告位 ID
-> 在代码中接入广告组件
-> 提交审核
-> 上线展示广告
```

常见广告组件包括 Banner、插屏、激励广告、视频广告、原生模板广告、封面广告等，具体以微信公众平台后台和官方文档为准。

## 小程序广告接入门槛

常见门槛：

- 小程序已经发布。
- 满足微信流量主开通条件。
- 无严重违规记录。
- 完成财务资料。
- 创建广告位。
- 小程序正式版审核通过后展示广告。

过往常见门槛是累计独立访客达到一定数量，例如 1000 UV 级别。实际门槛可能会随平台政策变化，最终以微信公众平台后台显示为准。

## 小程序广告怎么算收益

微信小程序广告通常按有效广告曝光结算，并与流量主分成。

腾讯广告培训资料中提到，小程序广告组件按每千次有效广告曝光收费，并按单日广告收入流水一定比例与流量主分成；资料中给出的普通小程序分成口径为 50%。

可以用这个公式理解：

```text
预估收入 = 有效曝光次数 / 1000 × eCPM × 分成比例
```

实际收益会受到广告位类型、用户地区、行业广告主出价、展示场景、点击率、填充率、无效流量过滤、平台分成和结算规则影响。

## 广告位怎么放

广告位要放在有“停顿感”的地方。

适合：

- 查询结果页
- 任务完成页
- 页面切换
- 解锁功能
- 获取额外次数
- 获取模板下载
- 游戏复活或道具

不适合：

- 用户一打开就强插广告
- 正在填写表单时打断
- 支付前后误导点击
- 频繁遮挡主要功能
- 用广告伪装功能按钮

## 广告变现适合什么产品

适合：

- 工具使用频次高
- 免费用户多
- 单用户付费意愿弱
- 内容或查询页多
- 用户停留时间较长
- 广告不破坏核心体验

不太适合：

- B2B 高价值工具
- 需要强信任的产品
- 医疗、金融、隐私敏感产品
- 低频但高价值产品
- 付费转化远高于广告收益的产品

如果一个用户每月愿意付 19 元，通常不要为了几分钱广告收入破坏体验。
